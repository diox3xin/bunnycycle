**BunnyCycle** — расширение для SillyTavern, которое добавляет в ролевую игру реалистичную систему репродуктивной биологии персонажей. Расширение автоматически отслеживает менструальные циклы, фертильность, беременность, роды, детей, а также поддерживает AU-сеттинги вроде омегаверса с течкой и гоном.

### Что умеет:

- 🔴 **Менструальный цикл** — фазы, фертильность, симптомы, либидо
- 🤰 **Беременность** — понедельное отслеживание, триместры, размер плода, симптомы
- 🏥 **Роды** — стадии от латентной до рождения плаценты
- 👶 **Дети** — карточки детей с возрастом, внешностью, родителями
- 🎲 **Dice roll** — бросок на зачатие с учётом фертильности и контрацепции
- 🔥 **Омегаверс** — течка, гон, супрессанты, вторичный пол
- 🥚 **Oviposition** — вынашивание, откладывание, инкубация яиц (не спрашивайте...)
- 💞 **Связи** — семейные отношения между персонажами
- 🧠 **AI-анализ** — нейросеть сама определяет пол, глаза, волосы и события из чата
- 📊 **Виджет** — статус персонажей в конце каждого сообщения AI
- ⏰ **Авто-время** — расширение само ловит временные скачки в тексте
- 💉 **Промпт-инъекция** — AI знает состояние персонажей и отыгрывает это

## Как пользоваться

### Первый запуск

1. **Открой чат** с каким-нибудь персонажем
2. Разверни панель **🐰 BunnyCycle** в настройках расширений (правая панель)
3. Нажми кнопку **🔄 Синхр.** — расширение найдёт персонажа и твою персону
4. Если включён **🧠 AI-анализ** — нейросеть автоматически определит пол, цвет глаз и волос из карточки

### Вкладки

| Вкладка | Что делает |

| **🔴 Цикл** | Календарь менструального цикла. Показывает фазу, фертильность, симптомы. Можно вручную переключить фазу или день |
| **🔥 Течка** | Омегаверс: управление течкой и гоном. Работает только при включённом AU оверлее с пресетом "Омегаверс" |
| **💕 Интим** | Ручная запись секс-сцен. Кнопка "🎲 Бросок" — dice roll на зачатие |
| **🤰 Берем** | Отслеживание беременности: неделя, триместр, размер, симптомы. Кнопки: +1 неделя, перевод в роды, прерывание |
| **🏥 Роды** | Управление родами по стадиям. Кнопка "👶 Родить" открывает форму создания ребёнка |
| **👶 Дети** | Список детей. Создание, редактирование, удаление |
| **🥚 Яйца** | Oviposition модуль (при включении) |

### Автоматический режим

Если в настройках включены автоматические функции, расширение само будет:

- **Синхронизировать** персонажей при загрузке чата
- **Определять пол и внешность** из карточки через AI
- **Ловить секс-сцены** в тексте AI (нужно минимум 3 сильных маркера: "проник", "кончил внутрь", "трахал" и т.п.)
- **Бросать кубик** на зачатие при вагинальном сексе с эякуляцией внутрь
- **Ловить временные скачки** ("прошло 3 дня", "через неделю", "на следующее утро") и продвигать циклы
- **Показывать виджет** с информацией о персонажах в конце каждого ответа AI
- **Инжектить промпт** чтобы AI знал о цикле, беременности, течке и т.д.

### Ручной режим

Если не доверяешь автоматике, можно всё делать вручную:

1. **Отключи** ненужные галочки в ⚙️ Настройках
2. Заходи в нужные вкладки и управляй кнопками
3. Нажимай **🎲 Бросок** в "Интим" для ручного dice roll
4. Продвигай время кнопками **+1 день** / **+7 дней**

### Редактирование персонажа

1. Вкладка **👥 Перс** → нажми ✏️ рядом с именем
2. Откроется редактор: пол, вторичный пол, раса, контрацепция, глаза, волосы, длина цикла, нерегулярность
3. Нажми **💾 Сохранить**
4. После ручного редактирования расширение не будет перезаписывать эти данные автоматикой

### Dice Roll (бросок на зачатие)

Шанс зависит от:
- **Фазы цикла** — овуляция ~25%, фолликулярная ~8%, лютеиновая ~2%
- **Контрацепции** — презерватив снижает на 85%, таблетки на 91%, ВМС на 99%
- **Течки** — в омегаверсе течка добавляет +35% к шансу
- **Типа секса** — анальный и оральный = 0% шанс
- **Эякуляции** — наружу = шанс × 0.05

Попап покажет результат. Можно принять, перебросить или отменить.

### AU: Омегаверс

1. Включи **🌐 AU Оверлей** в настройках
2. Выбери пресет **Омегаверс**
3. Задай вторичный пол персонажам (α/β/Ω) через редактор
4. Омеги получат систему течки, альфы — гона
5. Мужчины-омеги могут забеременеть (настраивается)

### Сохранение данных

- Данные **автоматически** привязаны к текущему чату
- При переключении чата загружается нужный профиль
- Кнопка **💾 Сохр.** в "Проф" — ручное сохранение
- **📤 Экспорт** — скачивает JSON файл со всеми данными
- **📥 Импорт** — загружает данные из JSON файла
- **🗑️ Сброс** — полная очистка всех данных
