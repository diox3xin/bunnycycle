# 🐰 BunnyCycle v3.0

**Комплексная система физиологии, здоровья и AU для SillyTavern**

## Возможности

### 🔴 Менструальный цикл
- 4 фазы: менструация → фолликулярная → овуляция → лютеиновая
- Фертильность, либидо, симптомы, выделения
- Визуальный календарь цикла
- Настройка длины цикла (21–45 дней)

### 🤰 Беременность
- 40-недельная прогрессия (настраиваемо)
- 3 триместра с симптомами
- Размер плода (от макового зерна до арбуза)
- Осложнения (15+ типов)
- Шевеления, набор веса, рост живота
- Многоплодная беременность

### 🏥 Роды
- 6 стадий: латентная → активная → переходная → потуги → рождение → плацента
- Раскрытие шейки, схватки, боль
- Осложнения (12+ типов)
- Создание ребёнка при рождении

### ❤️ Система здоровья
- **60+ болезней** в 6 категориях
- **Травмы** с локализацией, инфицированием, заживлением
- **Лекарства** с курсом лечения
- **Ментальное здоровье**: ПТСР, депрессия, панические атаки, выгорание
- **Иммунитет**, стресс, энергия, боль, кровопотеря
- Автогенерация болезней по контексту РП
- История болезней

### 🎲 Интимность и зачатие
- Детектор секса в тексте (regex, 20+ паттернов)
- Определение типа контакта, эякуляции, презерватива
- Кубик зачатия с модификаторами
- Контрацепция: презерватив, ОК, ВМС, ППА
- Лог интимности и бросков

### 🐺 AU: Омегаверс
- Альфа / Бета / Омега
- Течка и гон с фазами
- Связь (bond) + ломка
- Супрессанты
- Модификация беременности

### 🧝 AU: Фэнтези
- Расы с уникальными сроками беременности
- Ускоренная беременность
- Магическое лечение

### 🥚 Овипозиция
- Гестация → Кладка → Инкубация → Вылупление
- Настройка количества яиц

### 👶 Дети
- Вехи развития (15 этапов)
- Кривая веса
- Наследование внешности

### ⏰ Время мира
- Автопарсинг времени из текста
- Заморозка времени
- Прогрессия всех систем

### 🎭 Режиссура
- Автоматические подсказки AI по контексту
- Инъекция состояний в промпт

## Установка

1. Скопируйте папку `bunnycycle` в `SillyTavern/public/scripts/extensions/third_party/`
2. Перезапустите SillyTavern
3. Включите расширение в настройках

## Структура

```
bunnycycle/
├── index.js                    # Точка входа
├── manifest.json               # Манифест
├── core/
│   ├── stateManager.js         # Состояние, настройки, профили
│   ├── cycleEngine.js          # Менструальный цикл
│   ├── pregnancyEngine.js      # Беременность
│   ├── laborEngine.js          # Роды
│   ├── healthSystem.js         # Здоровье (60+ болезней)
│   ├── babyManager.js          # Дети
│   ├── auEngine.js             # AU (омегаверс, фэнтези, овипозиция)
│   ├── intimacyDetector.js     # Детектор секса + кубик
│   ├── timeManager.js          # Время мира
│   ├── promptBuilder.js        # Генератор промпта
│   ├── profileManager.js       # Менеджер профилей
│   └── relationshipManager.js  # Отношения
├── ui/
│   ├── drawerUI.js             # Рендеринг drawer
│   ├── widgetRenderer.js       # Виджет в чате
│   └── popupManager.js         # Попапы и диалоги
├── utils/
│   ├── helpers.js              # Утилиты
│   └── llmCaller.js            # Вызов LLM
└── assets/
    ├── templates/
    │   ├── drawer.html         # HTML drawer
    │   └── messageWidget.html  # Шаблон виджета
    └── styles/
        └── main.css            # Стили
```
